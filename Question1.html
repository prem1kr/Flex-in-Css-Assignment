Css Flexbox

1. Flexbox is a single diamensional layout.

2. It is useful in allocating and aligning the space among items in a grid container

3. To start using Flexbox, you have to create a flex container using the display: flex property.

4. Flexbox is great for layouts where the size of the content dictates the layout.

5. Flexbox is ideal for managing the layout of items within a single container.

Grid Layout

1. Grid is a two diamensional layout.

2. It is useful in creating more complex and organized layouts.

3. To define a grid container, you will have to pass a display: grid property to your element.

4. Grid is suitable for layouts where the design requires a defined structure.

5. Grid is ideal for managing the layout of entire web pages or large sections where both row and column arrangements are needed.
